02 EXPERIMENTS_CHARACTERISTICS
name description unit
id This is a unique identificator for each downloaded document, the first letter indicates the base which the documents was downloaded from (E for embrapa and S for Scopus). index
production_system Abbreviation of the production system in the experiment: ICLF = Integrated Crop-Livestock-Forestry; ICL = Integrated Crop-Livestock; ILF = Integrated Livestock-Forestry; C = Crop; L = Livestock; F = Forestry; EG = Extensive Grazing; SF = Seasonal Forest. nominal category
plot_id Unique experiment plot indentificator within the same publication (not unique across publications). index
plot_area The area of the experimental plot. hectar (ha)
total_area The total area of the experiment (all plots together). hectar (ha)
forest_component The species used in the production system that are part of the forestry component (woody species). This variable can be composed of one or more species, separated by comma. Species identified by scientific name. nominal category
crop_component The species used in the production system that are part of the crop component. This variable can be composed of one or more species, separated by comma. Species identified by scientific name. nominal category
pasture_component The species used in the production system that are part of the pasture component. This variable can be composed of one or more species, separated by comma. Species identified by scientific name. nominal category
other_component The species used in the production system that are not part of any of the components. This variable can be composed of one or more species, separated by comma. Species identified by scientific name. nominal category
livestock_component The species used in the production system that are part of livestock component. This variable can be composed of one or more species, separated by comma. Species identified by common name. nominal category
min_stocking_rate The minimum number of animals grazing in one hectare of pasture in the experiment or experimental plot. float
stocking_rate Number of animals grazing in one hectare of pasture in the experiment or experimental plot. float
max_stocking_rate The maximum number of animals grazing in one hectare of pasture in the experiment or experimental plot. float
integration_type Describe how the species are integrated in an experimental plot. Can be rotation, consortium or succession. One experimental plot can have different integration techniques, those are separated by a comma. nominal category
integration_obs Pertinent observations about the integration techniques used in an experimental plot. text
trees_spacing The spacing between trees adopted in an experimental plot. The spacing is represented by meters between trees. nominal category
trees_population The amount of trees individuals in one hectare of production. integer
soil_management Brief description of any relevant practice in soil management. text
previous_use What was the land use before the adoption of the production system analysed in the experiment. nominal category
establishment What was the year of the adoption of the production system analysed in the experiment. year according to the gregorian calendar
experiment_period Which years of the start and end of the experiment. The period is represented by two years, separated by a line. years according to the gregorian calendar
federative_unit The federative unit inside Brazil where the experimental plot was established. nominal category
municipality The municipality inside Brazil where the experimental plot was established. nominal category
experiment_location The specific location (e.g. farm name, institution unit) where the experimental plot was established. nominal category
coordinates The geographic coordinates provided to indicate the location of the experimental plot. text
soil_type_sibcs The soil type of the of the experimental plot(s), according to the Brazilian Soil Classification System. nominal category
soil_type_wrb The soil type of the of the experimental plot(s), according to the World Reference Base. nominal category
annual_precip Average historical accumulated precipitation in a year, for the experimental plot. milimeters (mm)
min_temp Average historical minimum temperature in a year, for the experimental plot. celsius degrees (ºC)
mean_temp Average historical mean temperature in a year, for the experimental plot. celsius degrees (ºC)
max_temp Average historical maximum temperature in a year, for the experimental plot. celsius degrees (ºC)
